Código Fonte / Pré-formatação

Es texto tem monoescrita, ou seja, letras ocupando o mesmo espaçamento, tornando-se mais fáceis de ser lidas. Ex.: um código element.systemprintline.io, com a tag CODE.

Para códigos de progamação digitados, usa-se a tag PRE, dessa maneira, o texta se apresenta igual no Visual Studio Code. E.: 

        int num = 1
        if (num<=2)
            print (esse número é num)
        else
            print (esse número é maior que num)
